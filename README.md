# Bangladesh Bank - Currency Note Management System

## About This Project
A web application for managing mutilated currency notes across 50+ branches.

## How It Works
1. **Inputter** adds note details
2. **Authorizer** reviews and approves
3. **Main Branch** does final verification

## Technology Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
- Database: MySQL

## Setup Instructions
1. Install Node.js
2. Install XAMPP (for MySQL)
3. Clone this repository
4. Run: npm install
5. Run: npm start
